# Anti-SEO Researcher

**反SEO反水军 深度消费决策研究工具**

> 在搜索结果被SEO和水军严重污染的中文互联网中，帮你找到真实的消费反馈，做出靠谱的购买决策。

## 快速开始

### 安装

将整个目录拷贝到你的 AI 编码助手的 Skill 目录下：

```bash
# CodeBuddy
cp -r anti-seo-researcher/ your-project/.codebuddy/skills/

# Cursor
cp -r anti-seo-researcher/ your-project/.cursor/skills/
```

或者通过 Git Clone：

```bash
git clone https://github.com/AmosHc/ai-skill-anti-seo-researcher.git
```

环境要求：Python >= 3.6（仅使用标准库，无需 pip install），以及一个支持 Skill 的 AI 编码助手（需提供 `web_search` 和 `web_fetch` 工具）。

### 使用

安装完成后，在 AI 助手的对话中直接描述你的购买需求：

```
帮我调研一下3000元以内的人体工学椅，主要打游戏用，每天坐6小时以上

想给宝宝买奶粉，预算300-350一罐，6个月大，主要关注安全性和消化吸收

预算5000左右的手机，主要拍照和日常使用，不玩游戏
```

工具会自动完成从搜索、过滤、评分到生成报告的全部流程，最终输出一份 Markdown 格式的调研报告，包含推荐/避坑结论、评分表、真实用户引用和数据来源。

## 它解决什么问题

当你想买一个商品时，搜索引擎给你的大多是知乎的品牌软文、小红书的广告合作、什么值得买的返利推荐、B站的品牌赞助测评。而真正有价值的消费反馈藏在京东淘宝的追评（用了半年后的真实感受）、小红书种草帖下面的评论区（"我也买了，翻车了"）、V2EX/Chiphell 等小众社区的长文吐槽、知乎回答下面的反驳评论里。

这个工具做的事情就是：**自动从中文互联网的各个角落搜集真实用户反馈，用算法过滤掉软文和水军，对候选商品进行多维度评分，最终生成一份带有推荐/避坑结论的调查报告。**

## 核心能力

### 三层数据源搜索

| 层级 | 来源 | 价值 |
|------|------|------|
| 电商追评 | 京东/淘宝/拼多多评论（间接搜索） | 花了真金白银的买家，长期使用后的真实追评 |
| 评论区 | 小红书/知乎评论区（间接搜索） | 种草帖下面的"拔草"评论，反驳软文的真实声音 |
| 社区帖 | V2EX / Chiphell / NGA / 贴吧等 | 发烧友的深度体验帖和横向对比 |

电商评论和社交评论区是动态加载内容，搜索引擎无法直接索引。工具通过**间接搜索策略**（搜索"追评汇总"、"差评合集"、"评论搬运帖"等二手内容）来获取这些高价值数据。

### 双层可信度过滤

第一层是正则规则引擎，100+ 条规则识别软文特征（营销话术、零缺点描述、购买链接、好评返现等）和真实反馈特征（使用时长、缺点描述、售后经历等）。第二层是 AI 语义分析，对正则评分处于灰色地带的内容调用 AI 做 5 维度精细判断（语气真实性、论述逻辑、细节丰富度、情感一致性、利益关联）。两层结合，既保证了过滤精度，又节约了约 40-50% 的 AI 调用成本。

### 品类自适应

不同品类的消费决策关注点完全不同。工具会根据你的需求自动生成品类配置，驱动后续所有步骤的差异化行为。比如奶粉关注配方成分和安全召回，安全封顶阈值最严格；手机关注 SOC 性能、续航和发热；人体工学椅关注久坐体验和网布耐久性；防晒霜关注成分安全性和肤感。

### 安全事件筛查

对每个候选品牌执行安全事件专项搜索，覆盖召回、致死、下架、诉讼、监管处罚等通用安全层，以及品类专属的安全风险（如奶粉的三聚氰胺、电子产品的电池爆炸、椅子的气压棒爆炸）。存在严重安全问题的商品会被强制降分，无论其他维度评分多高。

## 工作流程

整个调研分为以下步骤：

| 步骤 | 做什么 |
|------|--------|
| 需求确认 | 确认品类、预算、使用场景 |
| 品类分析 | 自动生成品类配置，驱动后续所有差异化行为 |
| 三层搜索 | 电商追评 + 评论区 + 社区帖，覆盖 12+ 个平台 |
| 内容抓取 | 抓取有价值帖子的完整正文 |
| 参数提取 | 从内容中提取商品结构化参数 |
| 动态深挖 | 对高频型号执行负面长尾搜索 + 电商追评深挖 |
| 安全搜索 | 对候选品牌搜索安全事件 |
| 可信度评估 | 正则粗筛 + AI 精判，过滤软文和水军 |
| 冲突仲裁 | 不同来源矛盾评价的加权裁决 |
| 多维评分 | 按品类维度打分 + 安全封顶 |
| 报告生成 | 输出 Markdown 调研报告 |

## 评分体系

### 可信度评分

每条搜索结果获得 0-100 的可信度评分。85 分以上为高度可信（长期使用反馈、有具体缺点），60-84 分基本可信，30-59 分进入灰色地带触发 AI 精判，30 分以下疑似软文直接过滤。

### 商品综合评分

| 综合分 | 结论 |
|--------|------|
| >= 70 | 推荐 |
| 55-69 | 有条件推荐 |
| 40-54 | 谨慎选择 |
| < 40 | 避坑 |

### 安全封顶

即使商品在其他维度评分很高，存在安全问题时综合分会被强制封顶。食品/奶粉封顶 30 分（最严格），个护/美妆封顶 25 分，电子产品封顶 20 分，耐用品封顶 15 分。

## 覆盖范围

搜索平台包括：V2EX、Chiphell、NGA、什么值得买、知乎、百度贴吧、小红书、豆瓣、B站、京东（间接）、淘宝（间接）、拼多多（间接）。

工具支持任意消费品类（通过品类配置自适应），已经过实际验证的品类包括：电竞椅/人体工学椅、婴儿奶粉、手机、显示器、键盘/耳机、床垫、空调/洗衣机/冰箱、防晒霜/美妆、保健品/食品。

## 示例输出

最终输出的调研报告是一个 Markdown 文件，包含：需求回顾、安全风险警告（如有）、候选商品评分表（动态表头，由品类维度决定）、调查结论（推荐/避坑/中性，每个结论附真实用户引用和来源链接）、数据源分布统计、数据透明度信息、免责声明。
