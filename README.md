# Anti-SEO Researcher

**Anti-SEO, Anti-Astroturfing Deep Consumer Research Tool**

> In an internet where search results are heavily polluted by SEO manipulation and astroturfing, this tool helps you find genuine consumer feedback and make informed purchase decisions — in any language, for any market.

## Quick Start

### Install

Copy the entire directory into your AI coding assistant's Skill folder:

```bash
# CodeBuddy
cp -r anti-seo-researcher/ your-project/.codebuddy/skills/

# Cursor
cp -r anti-seo-researcher/ your-project/.cursor/skills/
```

Or clone via Git:

```bash
git clone https://github.com/AmosHc/ai-skill-anti-seo-researcher.git
```

Requirements: Python >= 3.6 (standard library only, no pip install needed), and an AI coding assistant that supports Skills (must provide `web_search` and `web_fetch` tools).

### Usage

After installation, simply describe your purchase needs in natural language. The tool automatically detects your language and adapts to the appropriate regional platforms and search strategies.

**English:**

```
Research the best ergonomic office chairs under $500, I work from home and sit 8+ hours daily

Looking for baby formula for my 6-month-old, budget around $30-40, mainly concerned about safety and digestion

Best noise-cancelling headphones under $300 for commuting and office use
```

```
赤ちゃん用のミルクを探しています。予算は2000-3000円、安全性重視
```

The tool automatically handles the entire pipeline — from searching, filtering, scoring, to generating a final research report in your language with recommend/avoid conclusions, scoring tables, real user quotes, and data sources.

## What Problem Does It Solve

When you want to buy a product, search engines give you mostly sponsored content: paid Q&A answers, influencer collaborations, affiliate review blogs, and brand-sponsored "reviews". The genuinely valuable consumer feedback is hidden in e-commerce follow-up reviews (how a product holds up after 6 months of real use), comment sections under promotional posts ("I bought it too, and it was a disaster"), niche enthusiast forums with detailed complaints, and rebuttals in Q&A comment threads.

This tool **automatically searches real user feedback from across the internet in your market, uses algorithms to filter out ads and astroturfing, scores candidate products on multiple dimensions, and generates a research report with recommend/avoid conclusions.**

## Core Capabilities

### Three-tier Data Source Search

| Tier | Source | Value |
|------|--------|-------|
| E-commerce Reviews | Real buyer reviews from regional e-commerce platforms (indirect search) | Buyers who spent real money, long-term follow-up reviews |
| Comment Sections | Comment sections on social platforms (indirect search) | Real "debunking" feedback under promotional content |
| Forum Posts | Regional community forums and enthusiast sites | In-depth experience posts and comparisons |

E-commerce reviews and social comment sections are dynamically loaded content that search engines cannot directly index. The tool uses **indirect search strategies** (searching for "review compilations", "negative review roundups", "follow-up review summaries") to access this high-value data.

### Dual-layer Credibility Filtering

The first layer is a regex rule engine with 100+ rules identifying marketing signals (promotional language, zero-defect descriptions, affiliate links, incentivized reviews) and authenticity signals (usage duration, defect descriptions, after-sales experience). The second layer is AI semantic analysis for content in the gray zone, evaluating 5 dimensions: tone authenticity, narrative logic, detail richness, emotional consistency, and interest disclosure. This dual-layer approach maintains filtering precision while saving approximately 40-50% of AI call costs.

### Category Adaptation

Different product categories have completely different consumer decision factors. The tool automatically generates a category-specific configuration based on your needs, driving all differentiated behavior in subsequent steps. For example, baby formula focuses on ingredient safety and recall events with the strictest safety thresholds; phones focus on SoC performance, battery life, and thermals; office chairs focus on long-session comfort and mesh durability.

### Safety Event Screening

Every candidate brand undergoes a dedicated safety event search, covering recalls, fatalities, bans, lawsuits, regulatory penalties (using the appropriate regulatory bodies for your region), and category-specific safety risks. Products with serious safety issues are force-downgraded regardless of other dimension scores.

## Workflow

| Step | What It Does |
|------|-------------|
| Requirements Confirmation | Confirm category, budget, use case |
| Category Analysis | Auto-generate category + region config, driving all subsequent behavior |
| Three-tier Search | E-commerce reviews + comment sections + forum posts, covering regional platforms |
| Content Fetching | Fetch full text of valuable posts |
| Parameter Extraction | Extract structured product parameters from content |
| Dynamic Deep Dive | Negative long-tail search + e-commerce review deep dive for high-frequency models |
| Safety Search | Search safety events for candidate brands |
| Credibility Assessment | Regex pre-filter + AI deep analysis, filtering ads and astroturfing |
| Conflict Arbitration | Weighted resolution of contradictory reviews from different sources |
| Multi-dimensional Scoring | Score by category dimensions + safety capping |
| Report Generation | Output research report in user's language |

## Scoring System

### Credibility Scoring

Each search result receives a 0-100 credibility score. 85+ is highly credible (long-term feedback with specific defects), 60-84 is generally credible, 30-59 enters the gray zone triggering AI analysis, below 30 is suspected astroturfing and filtered out.

### Product Overall Scoring

| Score | Verdict |
|-------|---------|
| >= 70 | Recommend |
| 55-69 | Conditional Recommend |
| 40-54 | Caution |
| < 40 | Avoid |

### Safety Capping

Even if a product scores highly on other dimensions, safety issues force-cap the overall score. Food/baby products are capped at 30 (strictest), personal care at 25, electronics at 20, durable goods at 15.
